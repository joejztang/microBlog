This project so far is an almost copy of The Flask Mega Tutorial

### how to run

`export FLASK_APP=microBlog.py`

`flask run`

### Modules used in this project

* Flask-WTF

* flask-sqlalchemy

* flask-migrate

* flask-login

* flask-mail

* pyjwt

* flask-bootstrap

* flask-moment

* flask-babel

* guess-language_spirit
